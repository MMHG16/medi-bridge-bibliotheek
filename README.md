# Bibliotheek — Medi-Bridge Reference Library

Een Zotero-achtige referentiebibliotheek geintegreerd met het Medi-Bridge design system. Bevat 502 academische papers verdeeld over 5 onderzoeksgebieden, met volledige zoek-, filter- en sorteermogelijkheden.

## Deployment

- **Live URL**: https://gnehfuknybluq.kimi.page
- **GitHub**: https://github.com/MMHG16/medi-bridge-bibliotheek

## Features

- **502 publicaties** in 5 collecties
- **200+ gekleurde tags** voor snel filteren
- **Drie-panel layout**: collecties (links) | tabel (midden) | detail (rechts)
- **Zoeken & filteren** op titel, auteur, tags
- **Sorteren** op titel, auteur, jaar
- **Paginering** (25 per pagina)
- **Instellingen** pagina met export-opties
- **Over** pagina met statistieken en documentatie
- **Framer Motion** animaties
- **Volledig in het Nederlands (nl-NL)**

## Design System

- **Kleuren**: warm paper (#f4f1ed), teal (#3d8c87), cream (#faf8f5)
- **Typografie**: DM Serif Display (headings), DM Sans (body)
- **Stijl**: warm, editorial, calm-clinical
- **Geinspireerd op**: Medi-Bridge / Maai Care design system

## Tech Stack

- React 19 + TypeScript
- Vite 7.2.4
- Tailwind CSS 3.4.19
- shadcn/ui
- Framer Motion
- Lucide React
